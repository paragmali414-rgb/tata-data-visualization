# 📊 Tata Data Visualization Project

This project is part of the Tata Forage Virtual Experience Program.

## 🔍 Overview
Analyzed an online retail dataset to generate business insights using Tableau.

## 📈 Key Insights
- Revenue shows strong growth towards year-end (seasonal trend)
- Top countries include Netherlands, Germany, and France
- A small group of customers contributes most revenue
- Sales demand is concentrated in Europe

- Tableau
- Excel
